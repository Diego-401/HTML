Lista ordenada

  1. Dentro de la etiqueta "ol" se una "li" para indicar un elemento de la lista.
  2. Manzana
  3. Pera
  4. Sandia

Lista desordenada

  • Dentro de la etiqueta "ul" se una "li" para indicar un elemento de la lista.
